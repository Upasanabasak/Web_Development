Responsive Two Column Layout

Introduction

This page shows a two-column layout on wider screens and a single column on small screens using one simple media query.

Shrink the window below 780px to see the columns stack.

Implementation

The grid uses grid-template-columns: 1fr 1fr;. A media query sets it to 1fr for narrow screens. Panels are plain white with a light border for clarity.

Minimal colors and fonts keep focus on structure.
