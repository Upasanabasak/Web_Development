Image

tag : img
attributes : src , alt , width , height
src : source -> we provide the path or the URL of the image
alt : alternative -> we provide the text to be displayed if the image is not found
width : width of the image
height : height of the image

Image Format

  • JPEG ( best for photographs with many colors)
  • PNG (best for graphics with transparent background e.g LOGO)
  • SVG (a vector format that can be scale infinitely , used for logo or icon)
  • WEBP(have excellent compresion and quality )
